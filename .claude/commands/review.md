# 進捗レビュー

以下のファイルを読み込んで、学習進捗をレビューしてください：

1. `LEARNING_PLAN.md` - 学習計画（目標）
2. `PROGRESS.md` - 進捗管理（実績）
3. `docs/adr/` - 技術判断記録

## レビュー内容

1. **現在のフェーズと進捗率**
   - 現在どのPhaseにいるか
   - そのPhase内での完了率（✅の数 / 全タスク数）

2. **完了したタスク**
   - 最近完了したタスク一覧
   - 成果物の確認

3. **進行中のタスク**
   - 現在取り組んでいるタスク
   - ブロッカーがあれば特定

4. **次のマイルストーン**
   - 次に達成すべき重要なマイルストーン
   - 残りのタスク数

5. **懸念事項・提案**
   - スケジュールの遅れがあれば指摘
   - 改善提案

6. **ADRの状況**
   - 作成されたADR一覧
   - 重要な技術判断の記録漏れがないか確認

## 出力形式

簡潔なサマリーを日本語で出力してください。数値は具体的に示してください。
